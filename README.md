# LX Software Public Website

This repository hosts the LX Software public website built with Vite, React
Router, TanStack Query, and Bootstrap 5. Static assets are deployed to S3 and
served via CloudFront.

## Quick start

```bash
cd apps/public_www
npm install
npm run dev
```
